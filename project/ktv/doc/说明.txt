
	程序使用操作说明:

a   : 显示全部歌曲列表
w   : 选择上一个
s   : 选择下一个
d   : 删除
q   : 退出上一级
f   : 搜索
j   : 减小音量
k   : 增大音量
回车: 确定
空格: 播放与暂停

箭头左 : 滑动播放进度向左
箭头右 : 滑动播放进度向右
箭头上 : 播放上一首歌曲
箭头下 : 播放下一首歌曲

如何添加音乐:
进入之后, 按a键,这时出现全部歌曲列表, 使用w与s进行上下选择歌曲, 回车即可添加音乐至播放列表中
添加完成后按q退出添加歌曲功能

如何搜索音乐:
按f键之后, 后面就输入要搜索的大致歌曲名称, 按回车键即可搜索, 这时会出现搜索出的结果列表,使用
w与s键进行上下选择歌曲, 回车键即可添加音乐至播放列表中, 完成后按q返回

如何切换歌曲
方法一:
	添加音乐至播放列表后, 可以采用箭头上下键进行切换歌曲
方法二:
	添加音乐至播放列表后, 可以采用w与s进行选择要播放的歌曲, 然后回车即可播放

如何修改歌曲信息
在主页面按 m 键, 然后输入密码, 默认密码为123456
然后出现所有歌曲的列表, 按回车键对信息进行修改, 先是修改作者, 然后发布日期还有类型
修改完毕后可以继续修改或者按 q 键进行退出到播放页面
